# Changelog

All notable changes to the CodeCanvas extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-01-XX

### Added
- Initial release of CodeCanvas
- Background image selection from local storage
- Image URL support
- Opacity control (0-100%)
- Blur effects (0-30px)
- Brightness adjustment (0-200%)
- Contrast adjustment (0-200%)
- Saturation adjustment (0-200%)
- 9-position image alignment
- 5 sizing modes (cover, contain, original, stretch, tile)
- Multiple repeat modes
- Theme-aware opacity adjustment
- Dark/Light theme support
- Global and workspace-specific settings
- Persistent configuration
- Settings webview UI
- Command palette commands
- Status bar integration
- Recent images history
- Export/Import settings
- Settings profiles
- Full keyboard navigation support
- Accessibility features
- Comprehensive error handling
- Detailed logging

### Features
- Core background rendering engine
- Image caching for performance
- Real-time settings preview
- Lazy image loading
- Multi-format support (PNG, JPG, WEBP, GIF, SVG)
- GIF animation support
- Configuration management
- Theme detection and adjustment
- Workspace profiles

### Documentation
- README with quick start guide
- Command reference
- Configuration guide
- Troubleshooting section
- Contributing guidelines
